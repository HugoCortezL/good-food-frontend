<h2 id="running">Running the application 🎲</h2>

 
```bash
# Clone the repository
$ git clone https://github.com/HugoCortezL/good-food-frontend.git
 
# Go to folder
$ cd good-food-frontend
 
# Install the dependencies
$ npm install

# Start the application
$ npm run dev
```
